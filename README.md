# 文风工坊 - AI文学写作能力评测与优化系统

基于 RAG（检索增强生成）架构的八月长安文风分析与评测系统，支持风格段落语义检索、多模型文风改写对比、以及9维度写作能力量化评测。

> **项目定位**：将"文学写作好不好"拆解为可度量的能力维度，建立数据质量标准与评估体系，驱动模型文学写作能力持续提升。

## 在线体验

部署在 Streamlit Cloud，打开即用：[文风工坊 Demo](https://august-eight-rag.streamlit.app)

```
│   ├── 数据标注规范.md      # 数据质量标准与标注规范文档
│   └── 评测维度标准_v2.md   # 9维度评测标准详细定义
└── src/
    ├── __init__.py          # 包初始化
    ├── embedding.py         # 智谱API向量化 + JSON索引构建
    ├── retrieval.py         # numpy余弦相似度检索（支持分类/标签/情绪过滤）

```

## 核心功能

### 🔍 风格检索

- 输入自然语言描述，语义检索最匹配的八月长安风格段落
- 支持按 **分类**（5类）、**标签**（8个技法）、**情绪基调**（9种）三维过滤
- 搜索结果优先展示精华句（highlight），完整原文折叠

### ✍️ 风格改写

- 输入任意文字，系统自动检索风格参考段落作为 few-shot 示例
- 反套路 prompt 设计：明确规避万能忧伤模板、生理反应堆砌等常见问题
- 支持 **多模型对比**：GLM-4 / DeepSeek / Kimi / MiniMax 四模型改写对比
- 改写后可自动触发评测

### 📊 写作能力评测

将"写得好不好"拆解为 **9个可度量维度**（通用5维度 + 场景4维度），每个维度 1-5 分 + 简短理由：

**通用维度（必评）：**

| 维度 | 评测内容 |
|------|---------|
| 具象精准度 | 细节是否具体、意外、有生活质感 |
| 情感间接性 | 情绪是否藏在缝隙里而非表面 |
| 对话潜台词密度 | 对话是否有言外之意 |
| 克制留白程度 | 是否做到了"不写满" |
| 节奏控制 | 叙事节奏是否有效 |

**场景维度（按适用性评分，不适用可标N/A）：**

| 维度 | 评测内容 |
|------|---------|
| 环境映射质量 | 环境与心理是否错位而非直白对应 |
| 叙述者声音 | 是否有温度的旁观者视角 |
| 幽默反差运用 | 是否用幽默包裹沉重 |
| 青春期心理真实性 | 是否准确捕捉青春期特有的心理状态 |

评测结果包含：各维度评分 + 综合评分 + 总评 + 改进建议，支持 JSON 导出。

### 📖 创作工坊

- **人物档案**：角色设定、性格特征、成长弧线管理
- **关系图谱**：角色关系网络可视化
- **风格笔记**：写作技法积累与参考
- **小说存档**：本地稿件管理与版本对比

## 技术栈

- **LLM**：智谱 GLM-4 / GLM-4-flash（主力）+ DeepSeek / Kimi / MiniMax（对比模型）
- **Embedding**：智谱 embedding-3
- **向量检索**：numpy 余弦相似度（零编译依赖）
- **评测模型**：GLM-4（固定，确保评分一致性）
- **Web 界面**：Streamlit
- **数据格式**：JSON

## 数据标注规范

详见 `docs/数据标注规范.md`，包含：

- 7个标注字段的定义与规范
- 5个分类、8个技法标签、9种情绪基调的完整定义
- highlight 精华句标注原则
- 质量校验清单
