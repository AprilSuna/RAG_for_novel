# 八月长安文风数据标注规范 v1.0

> 本文档定义了文学文本摘录的数据质量标准与标注规范，用于支撑AI文学写作能力评测与优化系统。

---

## 一、数据采集标准

### 1.1 什么样的段落值得摘录？

✅ **值得收录：**
- 展现了八月长安某种独特写作技法的段落
- 情感表达方式有学习价值的段落（不是情感本身动人，而是"表达方式"动人）
- 能作为"好写作"标杆的段落——即面试官读了也觉得"这段写得确实好"
- 覆盖不同情绪基调：不仅限于遗憾/伤感，还需覆盖心动、欢喜、嫉妒、尴尬、释然等

❌ **不收录：**
- 纯情节推进、无技法展示的段落
- 只有情绪但没有独特表达方式的段落
- 过长的段落（建议控制在300字以内，超长的截取精华部分）

### 1.2 摘录长度规范
- 单段：50-300字
- 如摘录包含完整对话+环境描写，上限可放宽到500字
- 超过500字的段落，拆分为多条，各自独立标注

---

## 二、标注字段说明

每条摘录包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int | ✅ | 唯一编号，从1开始递增 |
| `text` | string | ✅ | 摘录原文，保留原始段落格式 |
| `category` | string | ✅ | 分类（见第三节，5选1） |
| `tags` | list[string] | ✅ | 写作技法标签（见第四节，每条1-3个） |
| `source` | string | ✅ | 来源作品名 |
| `scene` | string | ✅ | 一句话描述场景与技法用途（写给标注者自己看） |
| `highlight` | string | ✅ | 本段最值得学习的精华句（从原文中摘取，不是改写） |
| `emotion` | string | ✅ | 核心情绪基调（见第五节） |

---

## 三、分类体系（5类）

每个段落归入一个主分类：

| 分类 | 定义 | 典型特征 |
|------|------|---------|
| **心理与情感** | 以角色内心活动为核心的段落 | 心理独白、情绪觉察、回忆触发 |
| **对话与潜台词** | 以角色对话为核心的段落 | 说一套想一套、言外之意、潜台词层次 |
| **环境与氛围** | 以环境/场景描写为核心的段落 | 环境映射心理、氛围烘托、感官细节 |
| **叙事与节奏** | 以叙述方式和节奏控制为核心的段落 | 时空交错、高效叙事、节奏切换 |
| **幽默与反差** | 以幽默、自嘲、反差为核心的段落 | 自嘲、互怼、言行情境反差 |

---

## 四、写作技法标签（8个）

每条摘录标注1-3个最突出的技法标签：

| 标签 | 定义 | 判断标准 |
|------|------|---------|
| **环境映射心理** | 用环境细节外化角色心理状态 | 环境/天气/光线/声音与情绪有映射关系，但不是直白对应（难过≠阴天） |
| **对话潜台词** | 角色说的话与内心想法不一致 | 表面平淡/嬉笑，底层藏着更深的情感；需要读出"言外之意" |
| **克制留白** | 情感不写满，话不说透 | 最动人的部分是没说出口的；用"算了""不用说"等收束 |
| **幽默反差** | 用幽默/自嘲包裹沉重或尴尬 | 笑着说让人心疼的话；严肃场景突然插入吐槽 |
| **感官细节** | 用具体感官体验承载情绪 | 视觉/听觉/嗅觉/触觉的精准描写，细节"准"而非"美" |
| **时空交错** | 成年叙述者回望少年时光 | 叙述视角在现在与过去之间切换，带来温柔的感伤 |
| **情感转折** | 段落内情绪发生明显转向 | 从幽默到沉重、从平静到崩塌、从日常到悸动 |
| **场景氛围** | 营造特定场景的整体氛围感 | 不是单个细节，而是整个场景的"空气感" |

---

## 五、情绪基调（新增字段）

标注本段的核心情绪基调，用于按情绪检索和评测：

| 情绪基调 | 说明 |
|---------|------|
| 遗憾/错过 | 错过的机会、未说出口的话 |
| 心动/暗恋 | 暗中喜欢、小心翼翼的靠近 |
| 温暖/陪伴 | 安心的陪伴、默契的理解 |
| 尴尬/窘迫 | 社交中的尴尬、不知所措 |
| 释然/放下 | 终于接受、向前看 |
| 压抑/沉默 | 欲言又止、家庭/社交压力 |
| 惊喜/重逢 | 意外的相遇、猝不及防的触动 |
| 幽默/轻松 | 自嘲、互怼、日常趣味 |
| 伤感/失去 | 离别、失去、回不去 |

## 六、highlight标注规范

`highlight` 字段是本段"最值得学习的精华句"，标注原则：

1. **从原文中直接摘取**，不做任何改写
2. 优先选择**最能体现该段技法**的句子，而非情感最浓烈的句子
3. 如果精华是一个完整的短句，直接摘取
4. 如果精华需要上下文才能理解，可摘取2-3句（用"……"连接省略部分）
5. 长度控制在100字以内

**示例：**
- ✅ `"声音断在晚风里。"` —— 精准、克制，一句话承载了所有未说出口的话
- ✅ `"太阳早已不知踪影，可天还没有黑，冰激凌似的天空层层渲染，让人分不清头顶到底是什么颜色。"` —— 比喻意外，"分不清"本身就是情绪
- ❌ `"我失望极了。"` —— 这不是精华句，这是直白的情感词

## 七、新增摘录模板

在 `data/excerpts.json` 中追加新条目时，按以下模板填写：

```json
{
  "id": 15,
  "text": "（摘录原文，保留段落格式）",
  "category": "对话与潜台词",
  "tags": ["对话潜台词", "克制留白"],
  "source": "最好的我们",
  "scene": "一句话描述场景和技法用途",
  "highlight": "从原文中摘取的精华句",
  "emotion": "心动/暗恋"
}
```

## 八、数据扩展优先级

当前14段数据，按以下优先级补齐：

| 优先级 | 方向 | 当前数量 | 目标数量 | 原因 |
|--------|------|---------|---------|------|
| P0 | 对话与潜台词 | 3段 | 6-8段 | 八月长安最核心的技法，数据太少模型学不到 |
| P0 | 幽默与反差 | 2段 | 5-6段 | 独特风格标识，当前几乎无法支撑检索 |
| P1 | 心动/暗恋基调 | 2段 | 5-6段 | 当前偏遗憾/伤感，需补齐正向情绪 |
| P1 | 环境与氛围 | 3段 | 6-8段 | 环境映射是重要技法，需更多样例 |
| P2 | 叙事与节奏 | 3段 | 5-6段 | 已有覆盖，可适当补充 |
| P2 | 心理与情感 | 3段 | 5-6段 | 已有覆盖，可适当补充 |

**总目标：30-40段，覆盖所有分类和情绪基调。**

## 九、质量校验清单

每条新增摘录录入前，逐项检查：

- [ ] `text` 是否保留了原始段落格式？
- [ ] `category` 是否准确归入5类之一？
- [ ] `tags` 是否标注了1-3个最突出的技法？（不要贪多）
- [ ] `scene` 是否一句话说清了场景和技法用途？
- [ ] `highlight` 是否从原文中直接摘取？（不是改写）
- [ ] `emotion` 是否标注了核心情绪基调？
- [ ] 这段是否真的展现了八月长安的独特技法？（而非只是情节好）
