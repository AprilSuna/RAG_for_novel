# 文风工坊评测维度标准 v2

> 基于29段《最好的我们》摘录提炼，适用于八月长安风格及青春文学文本评测
> 2026-07-06 制定

---

## 评测框架说明

- **9个维度**，分为两层：
  - **通用维度（5个）**：每段必评，不可标记N/A —— 具象精准度、情感间接性、克制留白程度、节奏控制、青春期心理真实性
  - **场景维度（4个）**：可标记「不适用」，不参与计分 —— 对话潜台词密度、环境映射质量、叙述者声音、幽默反差运用
- 每个维度1-5分，N/A维度不计入综合评分
- 每个维度包含三层锚点：青春文学核心 → 八月长安技法 → AI典型错误
- 3分为及格线，5分为八月长安原文水平
- 评测使用GLM-4，temperature=0.3确保稳定性
- **N/A机制**：场景维度先判断适用性，不适用则标记N/A（如无对话→对话潜台词N/A，不涉及幽默→幽默反差N/A）。1分=「尝试了但失败」，N/A=「没涉及这个维度」

## 维度1：具象精准度

**定义**：细节是否具体、意外、有生活质感，且只属于此刻这个角色，而非通用套路。

**青春文学核心**：青春期的细节不是"美"，是"准"——尴尬的、不体面的、只有那个年纪才会在意的东西。青春文学最忌讳把青春期写成滤镜。

**子标准**：
- 意外性：不落套路，不使用万能意象
- 角色专属：只有这个人在这刻才会注意到这个细节
- 反浪漫化：敢于写不体面的真实

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 细节精准且意外，只属于此刻这个角色，不可替换。反浪漫化，敢于写不体面的真实。 |
| **4分** | 细节具体且有角色专属感，但意外性稍弱，或略有美化倾向。 |
| **3分** | 有具体细节但较为常见，缺乏意外感，或细节可适用于多个场景。 |
| **2分** | 细节模糊，接近"有画面但不具体"，偶尔使用套路意象。 |
| **1分** | 使用万能意象（灰天/冷咖啡/落叶/夕阳余晖），任何场景都能套，缺乏角色专属感。 |

**八月长安锚点**：
- "脸上有油光，额上有痘痘"（摘录1）——不体面但精准，只有愧疚视角下才注意到的真实
- "病房里弥漫着韭菜合子的味道……脸膛紫红的大爷身着病号服却敞着胸露着怀"（摘录9）——反浪漫化的医院
- "驴子埋着头，一边啪啪啪地撒了一路驴粪蛋"（摘录24）——用驴粪蛋而非"破败的乡村道路"

**AI典型错误**：用"灰天/冷咖啡/落叶"等万能意象；过度美化青春期（滤镜感）；细节可适用于任何场景。

## 维度2：情感间接性

**定义**：情绪是否藏在缝隙里，通过错位场景、不相关细节或叙事压缩间接传达，而非直接命名。

**青春文学核心**：青少年往往说不清自己什么感受，写作应该呈现这种"说不清"而非替代它命名。

**子标准**：
- 情感位移：角色把注意力转向不相关细节，情绪藏在偏移里
- 叙事压缩：用极简语言承载极重情绪
- 反命名：不使用"失望/难过/开心"等情感词

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 情绪完全间接表达，读者自己感受到而非被告知。使用情感位移或叙事压缩，效果精准且自然。 |
| **4分** | 大部分间接表达，但有一处略显刻意，或位移不够自然。 |
| **3分** | 部分间接表达，但仍有直白的情感词或解释性语句出现。 |
| **2分** | 情绪主要靠直接描写，间接手法仅偶尔使用且效果不佳。 |
| **1分** | 直接使用"失望/难过/开心"等情感词，或用大量比喻装饰情绪而非隐藏情绪。 |

**八月长安锚点**：
- "穿了这么多年，你为什么不换一件？"（摘录14）——久别重逢不说"想你"，注意T恤没换
- "后来就没有后来了。"（摘录13）——整段恋情终结压缩成7个字
- "但我感觉自己抢了人家的甜筒，还笑嘻嘻地蹲在墙角舔得正欢。"（摘录1）——用荒谬比喻替代愧疚

**AI典型错误**：直接写"她心里一阵酸楚"；用大量比喻装饰情绪（"心像被揉碎的纸"）；解释角色为什么有这个情绪。

## 维度3：对话潜台词密度

**定义**：对话是否有言外之意，角色是否"说一套想一套"。沉默、岔开话题、故意说反话都算潜台词。

**青春文学核心**：青春期的对话特征是"想说的说不出口，说出口的不是想说的"。

**子标准**：
- 沉默即对话：最重要的"台词"可能是没说出口的话
- 反话/装作不在意：用嘲讽、不屑、玩笑包裹真心
- 私密语言：只有特定关系才懂的暗号和仪式

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 对话充满潜台词，表面与底层形成张力。沉默、反话、暗号运用精准，角色"说一套想一套"自然可信。 |
| **4分** | 对话有潜台词层次，但某一处略显刻意，或反话意图过于明显。 |
| **3分** | 对话有信息但缺乏潜台词层次，角色基本"说什么想什么"。 |
| **2分** | 对话主要承担信息传递功能，偶尔有潜台词但不明显。 |
| **1分** | 对话直白表达情感，没有言外之意。（如无对话则评N/A） |

**八月长安锚点**：
- "他看了一眼，动动嘴唇，却什么都没说。"（摘录19）——最高密度潜台词是沉默
- "给你个面子而已。小爷哪儿用得着这种封建迷信。"（摘录23）——表面不屑，实则珍视仪式
- "我知道。"（摘录5）——两个字承载"我一直在等你说"和"但我现在不能回应"

**AI典型错误**：让角色直接说出感受；用过度戏剧化台词；不会用"说一半""说岔了""故意说反话"制造潜台词。

## 维度4：克制留白程度

**定义**：是否做到了"不写满"，最动人的部分是否是未说出口的。留白要让读者比角色更早意识到遗憾。

**青春文学核心**：青春期最动人的时刻往往是没发生的事——没说出口的告白、没赴的约、没追问的问题。

**子标准**：
- 读者补全度：好的留白让读者自己补全，而非作者点明
- 情绪止步：在情绪最高点收手，不继续解释"因为……所以"
- 信息差利用：叙述者知道结局但角色不知道，差距产生反讽

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 精准收束，留白处比写出的话更有力量。读者补全度高，情绪止步在最动人的位置。 |
| **4分** | 收束尚可，留白意图清晰，但某一处略有"多写一句"的冲动，或读者补全度稍低。 |
| **3分** | 收束尚可但略显完整，缺少"欲言又止"的余韵，或留白意图不够明显。 |
| **2分** | 情绪基本写满，偶有停顿但缺乏留白意识，读者无需参与补全。 |
| **1分** | 情绪完全写满，没有留白。在情绪高点继续解释，不信任读者的理解力。 |

**八月长安锚点**：
- "声音断在晚风里。" + "以后有的是机会，听你说那些没说出口的话。"（摘录5）——读者知道没有"以后"
- "但没有对他说'谢谢'，说不出口。"（摘录4）——停在这里，不解释为什么
- "后来就没有后来了。"（摘录13）——7个字结束一段恋情

**AI典型错误**：在情绪高点继续解释"因为……所以她没有说出口，内心充满了矛盾"；不信任读者理解力，把潜台词说破。

## 维度5：环境映射质量

**定义**：环境与心理的关系是否错位、间接、有想象力，而非直白对应。环境是记忆的容器，不是气氛的标签。

**青春文学核心**：校园环境（教室、走廊、操场、放学路）本身就承载青春期记忆，环境描写不是"烘托气氛"，是记忆的容器。

**子标准**：
- 拟人化映射：环境被赋予与角色情绪相关的"态度"（如路灯"怂恿"）
- 感知滤镜：同一环境因角色心情不同而呈现不同面貌
- 声音锚定：用环境声音替代沉默的尴尬或情绪的直白
- 反pathetic fallacy：不用"难过→阴天，开心→阳光"的直白对应

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 环境与情绪形成错位或张力，映射方式意外且精准。使用拟人化、感知滤镜或声音锚定等高级手法，环境成为记忆容器。 |
| **4分** | 环境描写有效且与情绪有映射关系，但手法较为常规，或映射方式不够意外。 |
| **3分** | 有环境描写但映射方式较为常规（如直接用天气对应心情），缺乏想象力。 |
| **2分** | 环境描写与情绪关联薄弱，或仅作为背景板存在。 |
| **1分** | 环境与情绪直白对应（难过→阴天），或无环境描写。（如本段不涉及环境则评N/A） |

**八月长安锚点**：
- "路灯在他背后用橙色的光芒明目张胆地怂恿着"（摘录5）——路灯拟人化为"怂恿者"
- "客厅里只有可乐罐里面的气泡争先恐后地破裂"（摘录2）——用气泡声替代沉默的尴尬
- "冰激凌似的天空层层渲染，让人分不清头顶到底是什么颜色"（摘录12）——模糊的天空映射模糊的心情
- "世界忽然就变成了彩色。"（摘录27）——感知滤镜，心情转变时世界"变色"

**AI典型错误**：pathetic fallacy（难过→阴天/雨，开心→阳光）；堆砌环境描写但与情绪无关；环境描写过于文学化而失真。

## 维度6：叙述者声音

**定义**：是否有温度的旁观者视角，带自嘲/温柔/心疼。青春文学的叙述者通常是回望者——站在未来看过去。

**青春文学核心**：青春文学的叙述者通常是回望者，有温柔的讽刺、事后诸葛的自嘲、和对当初自己的心疼。这个"回望距离"是青春文学独有的。

**子标准**：
- 回望距离：叙述者知道结局但角色不知道，差距产生反讽和温柔
- 自嘲即自慰：自嘲不是真的贬低，是给自己的安慰
- 流行文化引用：用通俗参照系拉近与读者的距离
- 金句提炼：用一句话总结复杂感受

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 叙述者有鲜明的态度和温度，自嘲或温柔恰到好处。回望距离运用精准，叙述者与角色的信息差产生反讽或心疼。有金句提炼。 |
| **4分** | 叙述者声音存在且鲜明，但回望距离运用不够，或自嘲/温柔稍显刻意。 |
| **3分** | 叙述者声音存在但不够鲜明，缺乏回望视角，或态度不够有温度。 |
| **2分** | 叙述者声音微弱，像中性记录，偶尔有态度但不稳定。 |
| **1分** | 叙述者像冷冰冰的摄像机，没有态度，没有温度，没有回望意识。 |

**八月长安锚点**：
- "关于我的故事，好像都发生在过去。"（摘录7）——一句话确立回望视角
- "你看我爸和他姑爷相处得多好啊。虽然现场只有我有足够的远见"（摘录15）——自嘲式脑补
- "恋爱这种事情就是这样，对无法置身其中的旁观者来说，它是如此恶心又动人。"（摘录11）——金句提炼
- "我会错觉自己误闯了《行尸走肉》的片场"（摘录14）——流行文化引用

**AI典型错误**：叙述者声音扁平化——全程严肃抒情或全程搞笑；缺乏自嘲；没有回望视角，叙述者与角色信息完全同步。

## 维度7：幽默反差运用

**定义**：是否用幽默/自嘲包裹沉重或尴尬，且幽默与情感不脱节。好的幽默反差有"后劲"——笑完发现其实很疼。

**青春文学核心**：青春期的幽默是防御机制——用自嘲、互怼、荒诞化来处理说不出口的尴尬和心动。

**子标准**：
- 反差后劲：笑完之后后劲上来，发现刚才那段其实很疼
- 幽默即结构：幽默不是点缀，是情感结构的一部分
- 自我指向：幽默主要指向自己（自嘲），而非嘲笑他人
- 荒诞真实：幽默基于具体、荒诞但真实的细节，不是段子或梗

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 幽默与沉重形成精准反差，笑着心疼。幽默是情感结构的一部分，有明确后劲。自我指向，基于荒诞真实。 |
| **4分** | 有幽默反差但后劲稍弱，或幽默与情感的衔接不够自然。 |
| **3分** | 有幽默元素但反差感不强，或幽默与情感各自独立、缺乏转化。 |
| **2分** | 幽默元素零星出现，与主线情感关联不大，或过于"梗化"。 |
| **1分** | 纯严肃或纯搞笑，无反差。（如本段不适合幽默则评N/A） |

**八月长安锚点**：
- "我正在心里撒了欢儿地意淫" → "忽然觉得现场一片安静"（摘录15）——自嘲脑补→被抓包尴尬，笑完意识到她多在意
- "趴在床边的我彻底石化了。" → "它是如此恶心又动人"（摘录11）——旁观甜蜜→孤独翻转
- "如果我是清风，我一定弄死心相印。"（摘录20）——荒诞但真实的叛逆幽默

**AI典型错误**：幽默和情感脱节——前面搞笑后面煽情，中间没有转化；幽默太"梗化"不像角色自然反应；嘲笑他人而非自嘲。

## 维度8：节奏控制

**定义**：叙事节奏是否有效，长短句配合、场景切换是否自然。青春文学要还原主观时间感——有些日子漫长如年，有些三年一晃而过。

**青春文学核心**：青春期的记忆不是匀速的，节奏控制要还原这种主观时间感。

**子标准**：
- 时间压缩：用一句话跳过大段时间，在关键瞬间极度缓慢
- 长短句配合：长句铺陈→短句重击，制造停顿感
- 场景呼吸：对话与环境观察交替出现，给读者喘息空间
- 情绪节拍：句长镜像情绪状态—— overwhelm时短促，反思时绵长

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 节奏精准，快慢交替自然，关键处有停顿感。时间压缩与慢镜头运用得当，长短句配合有效。 |
| **4分** | 节奏良好，但某一处快慢转换略显生硬，或时间压缩/慢镜头使用不够精准。 |
| **3分** | 节奏尚可但平铺直叙，缺少变化，或节奏意识存在但执行不够稳定。 |
| **2分** | 节奏单调，全程匀速，偶有变化但不明显。 |
| **1分** | 节奏混乱或全程平铺，无节奏意识。该快的地方啰嗦，该慢的地方一笔带过。 |

**八月长安锚点**：
- 长段叙述大学恋爱 → "后来就没有后来了。" → 跳到喝可乐的当下（摘录13）——极度压缩后突然拉回
- 长段恐怖铺垫 → "这个声音几乎把我的整个世界都按成了暂停。" → 缓慢辨认过程（摘录14）——先急停再慢镜头
- 长段铺垫 → "对不起。" → "余淮的脸抽了抽。"（摘录25）——长 buildup 后两个短句重击

**AI典型错误**：全程平铺直叙不会用短句制造停顿；节奏混乱——该快的地方啰嗦，该慢的地方一笔带过；缺乏时间压缩意识。

## 维度9：青春期心理真实性（通用维度·必评）

**定义**：角色反应是否符合那个年纪的心理特征——不过度成熟也不过度幼稚，捕捉青春期特有的「情绪强度极高但表达能力极弱」。

**青春文学核心**：这是唯一不关心「怎么写」而关心「写得对不对」的维度。青春期的核心特征：情绪强度极高（小事天塌），但表达能力极弱（想说的说不出口）。角色不是30岁的人装16岁，也不是刻意幼稚，而是有那个年纪特有的笨拙、冲动和「事后才明白」。

**子标准**：
- 情绪强度匹配：青春期的小事天塌感——一次没说话可能比一次分手更痛
- 表达笨拙感：想表达但表达不出，而非流畅精准的自我剖析
- 事后才明白：青春期往往是当下混沌、事后清醒，角色不应有即时的高度自我认知
- 年龄一致性：反应符合角色年龄，不超前也不退行

### 评分标准

| 分数 | 描述 |
|------|------|
| **5分** | 角色心理高度真实，情绪强度与表达能力之间的gap精准呈现。反应完全符合那个年纪——有笨拙、有冲动、有事后才明白的特征。捕捉到「小事天塌」感。 |
| **4分** | 角色心理基本真实，但某处反应稍显成熟或稍显幼稚，或情绪表达略显流畅（真实青少年往往更笨拙）。 |
| **3分** | 角色反应基本可信，但存在明显「成人视角代入」或「过度幼稚化」的倾向。情绪强度的呈现不够精准。 |
| **2分** | 角色心理明显失真，反应不像那个年纪的人——要么过于理性成熟，要么过于夸张幼稚。 |
| **1分** | 角色像披着青少年皮的成年人，心理反应完全不符合年龄特征。情绪表达过于流畅精准，毫无青春期的笨拙和混乱。 |

**八月长安锚点**：
- 「我正在心里撒了撒儿地意淫」（青春期特有的脑补和幻想，毫不掩饰）
- 「我知道。」（两个字，想说的说不出口）
- 「但我感觉自己抢了人家的甜筒，还笑嘻嘻地蹲在墙角舔得正欢。」（用荒谬比喻替代愧疚，是那个年纪才会有的自嘲方式）

**AI典型错误**：角色心理过于成熟——16岁的人有着30岁的洞察力和自我认知；情绪表达过于流畅精准；用成人视角解释青少年行为（「她其实是因为童年创伤导致……」）；缺少「事后才明白」的特征——青春期往往是当下混沌、事后清醒。

## 场景自适应权重（未来迭代方向）

不同场景类型应权重不同：

| 场景类型 | 高权重维度 | 低权重/可选维度 |
|----------|-----------|----------------|
| 心理描写 | 情感间接性、克制留白、具象精准度 | 对话潜台词(N/A)、幽默反差(可选) |
| 环境描写 | 环境映射质量、节奏控制、感官细节 | 对话潜台词(N/A)、叙述者声音 |
| 对话场景 | 对话潜台词密度、幽默反差、克制留白 | 环境映射(可选) |
| 叙事过渡 | 节奏控制、叙述者声音、时空交错 | 对话潜台词(N/A)、幽默反差(可选) |

当前版本已实现5档评分标准 + N/A机制（场景维度可标记不适用），场景自适应权重作为v3迭代方向。
